Interactive demos
=================

`Essentia.js browser demos <https://mtg.github.io/essentia.js/examples/#/demos/>`_ - real-time music analysis in the browser, including tagging, genre, mood, key, BPM, onset detection, audio metering, chroma, pitch, and mel-spectrograms.

`Replicate demos <https://replicate.com/mtg/>`_ - cloud-hosted inference demos for tagging, genre, mood, arousal/valence, approachability, engagement, and BPM.

`AcousticBrainz <https://acousticbrainz.org>`_ - a large-scale music analysis database with descriptors for millions of tracks.

`Cosine.club <https://cosine.club/>`_ - an interactive tool for exploring music similarity using the `Discogs-Effnet <models.html#discogs-effnet>`_ audio embeddings.
